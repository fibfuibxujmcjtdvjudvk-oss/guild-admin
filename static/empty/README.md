# 空状态图片资源

## 需要添加的图片

### no-footprint.png
- **用途**: "还没有足迹"空状态插画
- **推荐尺寸**: 400x400 像素
- **格式**: PNG
- **风格**: 简约、友好、彩色插画
- **内容建议**: 
  - 地球仪或地图相关元素
  - 行李箱、飞机、旅行相关图标
  - 人物探索的插画

**在线资源推荐**:
- https://undraw.co/illustrations (免费插画)
- https://storyset.com (免费动画插画)
- https://www.manypixels.co/gallery (免费插画库)

## 临时解决方案

如果暂时没有插画，可以：

1. **使用 Emoji 组合**
```vue
<view class="empty-state">
  <text style="font-size: 120rpx;">🌍✈️🧳</text>
  <text>还没有足迹记录</text>
</view>
```

2. **使用纯文字**
   去掉 `<image>` 标签，只保留文字提示

3. **使用图标字体**
   使用 uni-icons 或其他图标库
